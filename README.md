# SaaS管理アプリ

自分や組織が契約しているSaaSの契約・費用・利用状況を一元管理するアプリケーション

## 概要

複数のSaaSサービスの管理を効率化し、コスト削減と利用状況の可視化を実現します。

## 主な機能

- **契約情報管理**: プラン・料金・支払方法の管理
- **アカウント情報**: メール・URL・用途の管理
- **利用状況**: 使用頻度・最終ログイン日の追跡
- **課金管理**: 更新日・自動更新フラグの設定

## 特徴

- **セキュリティ重視**: パスワードは扱わない
- **コスト可視化**: 費用管理がメイン価値
- **サブスク削減**: 無駄なサブスク契約の発見と削減

## 技術ポイント

- CRUD（基本機能）
- 日付処理（更新通知）
- グラフ表示（コスト可視化）
